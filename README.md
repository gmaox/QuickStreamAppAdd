将工作目录所有快捷方式一键导入到sunshine应用程序并自动处理封面<p>

为什么要自己造前端？/优势：
<p>1.依托qt5的自适应布局，界面布局简单舒适
<p>2.一键启动。playnite等一众前端启动游戏时更倾向于展示游戏介绍信息甚至启动影片，更倾向于电玩店，自用其实不太需要。
<p>3.一键调整收藏游戏来固定游戏顺序，同时不影响游戏的启动顺序记录
<p>4.配置简单，更专注于游戏
<p>5.软件启动迅速，告别前摇
<p>0.依靠sunshine和qsaa管理游戏列表
